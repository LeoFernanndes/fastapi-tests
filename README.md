# Fast Api Tests

## Dependencies:
Python 3.9 (3.9.13)  
Docker 20.10 + docker-compose 2.12  

## Application setup
1. clone this repo.  
2. cd into it and create .env.docker-compose file following sample.env.docker-compose variables.  
3. run docker compose up --build.
